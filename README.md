PokeTools
=========

ポケモン用ツールいろいろ  

今後の予定
- CSV関連の処理を外部ライブラリ化する
 - 出来れば2次元配列として処理したい
 - 列や行の挿入とか
- アプリの分割
 - 各機能を独立したアプリにする

- 構造イメージ
 - 共通の部分をまとめたランチャーアプリ
  - ファイルパスの設定
   - 外部から閲覧できるようにすること
  - Dropboxとの連携管理
   - ファイル個人共有サービスならなんでもいい
 - 各機能を持った個別アプリ
  - 大きく分けて3つの機能を持つ
   - CSVファイルを読む
   - 計算結果を表示する
   - 一部の結果をCSVファイルに書き込む
  - これ以外の機能
   - カメラを用いた選出画面の識別
   - 画像の表示
